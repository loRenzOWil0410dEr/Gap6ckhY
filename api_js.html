Working with APIs

APIs (Application Programming Interfaces) allow communication between different software applications. JavaScript can be used to fetch data from APIs and dynamically update web pages.

Using Fetch API

The Fetch API provides a simple way to make HTTP requests.

fetch("https://api.example.com/data") .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error("Error:", error));

Using Async/Await with Fetch

Async/Await makes working with APIs more readable and manageable.

async function getData() { try { let response = await fetch("https://api.example.com/data"); let data = await response.json(); console.log(data); } catch (error) { console.error("Error:", error); } } getData();
